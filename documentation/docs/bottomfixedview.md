BottomFixedView extends from View only,and it helps any content under it to place at the bottom of the screen




## Usage

```jsx
import { BottomFixedView } from 'viserion';

<BottomFixedView>
    <View>
      <Text>Fixed at the bottom</Text>
    <View>
</BottomFixedView>
```
