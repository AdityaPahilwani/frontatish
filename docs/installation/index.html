Installation

info

The library is still under development,create issues here

yarn add frontatish
or
npm install frontatish

Requirements#

This library depends on some third party library like react-native-vector-icons , react-native-gesture-handler & react-native-reanimated

If you are at >0.60 react-native versions then you don't need to link these libraries or if you are at below to this version you would need to link it manually these libraries into your project.

react-native link <package-name>

Usage#

import React from 'react';
import { View } from 'react-native';
import { Button } from 'frontatish';
const App = () => {
return (
<View>
<Button label="PRESS ME!">
</View>
);
}
